Este respositório contém um projeto voltado para converter a base numéricas de numeros digitados pelo usuário. <br>
Foi desenvolvido como um projeto acadêmico.

Possui duas versões:
  - Python (desenvolvido por @nome) e
  - VBA para Excel (por @vilum).

Para executar a versão em Python:
  1. Certifique-se de ter o Python instalado;
  2. No terminal, execute 'python src.py'.

Para executar a versão em VBA:
  1. Certifique-se de ter o Excel instalado;
  2. Abre o arquivo 'conversor de bases.xlsm'
  3. Habilite macros no Excel;
  4. A interface da planilha permite inserir um número e converter entre diferentes bases.
